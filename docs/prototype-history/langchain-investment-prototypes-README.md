# LangChain Investment Prototypes

Dedicated scratch space for quick experiments related to the multi-agent LangChain investment research project.

This folder is intentionally separate from the main project so prototypes can move fast without adding churn to production code.

## Structure

- `prototypes/` - runnable experiments, small scripts, or throwaway proof-of-concepts.
- `notes/` - research notes, design sketches, prompts, and findings.
- `artifacts/` - generated transcripts, logs, reports, or model outputs.
- `data-samples/` - small sanitized sample inputs for experiments.

## Suggested Rules

- Keep each prototype self-contained in its own dated or named subfolder.
- Promote only proven ideas back into the main project.
- Avoid storing private credentials, raw proprietary data, or large media files here.
- Put assumptions and next steps in each prototype's local README.
